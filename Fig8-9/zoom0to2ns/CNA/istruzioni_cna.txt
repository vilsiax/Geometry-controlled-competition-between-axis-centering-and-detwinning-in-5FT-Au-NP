NOTA: Questo programma non fa minimizzazioni, ma prende una lista di file di tipo xyz, a singola componente o a due componenti, e ne calcola la mappa cna e le percentuali dei vari tipi di atomi

COMPILATION: use the Makefile, the executable is cna_v01

INPUT FILES
- files_per_cna.in: contains the list of the names of xyz files for which cna signatures have to be calculated. Coordinates must be in Angstrom.
- input_cna.in: contains the few parameters needed for the calculations, for example
si			!choicecnamaps (si/no)
si			!choicecnastats (si/no)
Ag,Ag			!momemet(1),nomemet(2) that are the symbols of the species
1.445,1.445		!rat(1),rat(2) that are the atomic radii of the species, in Angstron
0.20			!tolerance_nn, tolerance on the nearest-neighbour distance

comments on input files: 
choicecnamaps is to produce output xyz files with cna signatures for all atoms and corresponding colors; choicecnastats is to produce output txt files with all cna statistics and detailed signatures.
In the case of a binary system, the atomic radius for the mixed pairs is rat(3) which the average of rat(1) and rat(2). The cutoff distance for neighbours is 2*rat*(1+tolerance_nn).
 
OUTPUT FILES
given for example the following files_per_cna.in
pippo1.xyz
q.00001t0500
the program will produce the output files
pippo1.xyz_cna_map.xyz
q.00001t0500_cna_map.xyz
and/or
pippo1.xyz_cna_stats.txt
q.00001t0500_cna_stats.txt
